📊 Bank Marketing Dashboard | Microsoft Excel
📌 Project Overview

This project presents an interactive Bank Marketing Dashboard developed in Microsoft Excel using the Bank Marketing dataset. The dashboard provides insights into customer demographics, marketing campaign performance, and deposit subscription trends to support data-driven decision-making.

🎯 Objectives
Analyze customer demographics.
Monitor deposit subscription performance.
Identify customer behavior across age groups, education, and marital status.
Understand monthly contact trends.
Create an interactive dashboard for business reporting.

📂 Dataset
The dataset contains customer information collected during a bank marketing campaign.

📊 Key Insights
Most customers belong to the 30–39 age group.
Secondary education customers represent the largest customer segment.
Married customers form the majority of the customer base.
Deposit subscriptions show a nearly balanced acceptance and rejection ratio.
Customer contacts peaked during the month of May.
